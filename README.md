## Build Setup

Clone the repo

```
git clone https://github.com/sofi-bel/how-to-learn.git
```

Install dependencies

```
npm install
```

Server with hot reload at http://localhost:9000/

```
npm run start
```

Production Build

```
npm run build
```

Development mode

```
npm run dev
```
